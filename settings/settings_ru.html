🎬 Видео и 📸 скриншоты

Запись видео вкладки и сохранение скриншотов при воспроизведении теста. Файлы сохраняются в папку «Загрузки» в указанную подпапку (видео — в подпапку videos, скриншоты — в screenshots).

🎬 Видео

Без постобработки файл меньше; с ней — перемотка в любом плеере.

📸 Скриншоты

Доступно только при включенном сохранении скриншотов
Путь задаётся относительно папки «Загрузки». Будут созданы подпапки videos и screenshots; внутри — подпапка с ID теста и временем прогона

📁 Управление файлами для тестов

Загрузите файлы, которые будут использоваться в автотестах. Файлы сохраняются в расширении и могут быть автоматически вставлены в поля загрузки файлов.

Максимальный размер одного файла: 5MB. Общий лимит хранилища: ~10MB

📊 Экспорт тест-кейсов в Excel

По умолчанию отчёты доступны только через кнопку 📊 «Экспорт отчёта» в истории прогонов

Когда экспортировать:

Формат XLS сохраняет CSV с расширением .xls (Excel открывает без предупреждений). Выберите XLSX для полноценного файла Excel или CSV для чистого текста.
По умолчанию используется точка с запятой ; — смените при необходимости, например на запятую ,
Если не указан, файлы будут сохраняться в папку Загрузки браузера
