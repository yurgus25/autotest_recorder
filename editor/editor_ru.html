Редактирование действия

Добавить переменную

Латинские буквы, цифры и подчёркивание. Начинается с буквы.
Селектор:
Выделенная часть:
Позиция: символы - Примечание: позиции применяются к актуальному тексту элемента во время выполнения теста
Укажите позиции символов для извлечения части текста из элемента
При экспорте будет запрошено подтверждение

🔗 Извлечь переменные из URL

Для query: имя параметра. Для path: номер сегмента. Для regex: регулярное выражение с группой захвата.

📱 Отправка сообщения в Telegram

Токен вашего Telegram бота (можно получить у @BotFather)
ID пользователя или чата (можно узнать у @userinfobot)
Можно использовать переменные: {var:имя}, {date}, {time}, {datetime}, {counter:имя}
💡 Переменные будут автоматически сохранены:
• telegramBotToken (токен бота, чувствительные данные)
• telegramChatId (ID чата/пользователя)

🎯 Извлечение переменной со страницы

Выберите вкладку, на которой находится нужный элемент
Выбранный элемент:
Имя переменной для сохранения значения
Используйте группу захвата для извлечения части значения (например: ID: (\\d+))
Если URL страницы совпадает с URL в шагах теста, значение будет извлекаться автоматически при выполнении

📊 Управление переменными

Переменные всех сценариев

Переменные из всех сценариев. Переменные текущего сценария имеют приоритет при выполнении.

Переменные циклов

💡 Переменные циклов создаются автоматически при добавлении цикла.
При старте цикла переменная инициализируется в 0 и увеличивается на 1 после каждой итерации.
После завершения цикла переменная равна количеству выполненных итераций.

Переменные циклов управляются автоматически, их не нужно объявлять вручную.
