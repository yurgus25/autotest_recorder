📊 Manage variables

Variables from all scenarios

Variables from all scenarios. Current scenario variables have priority during execution.

Loop variables

💡 Loop variables are created automatically when adding a loop.
At the start of the loop, the variable is initialized to 0 and incremented by 1 after each iteration.
After the loop completes, the variable equals the number of iterations performed.

Loop variables are managed automatically, no need to declare them manually.
